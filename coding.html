Упражнение 2.1.0

В следующих упражнениях допущена ошибка, программа должна выводить строку "привет" ровно три раза
Предскажи сколько раз будет выведена строка, если запустить программу как есть,
Затем исправь код не ломая существующую структуру так, чтобы строка выводилась ровно три раза 


i = 0
while i < 4:
    print('Всё работает как надо')
    i = i + 1
показать ответ
Строка будет выведена 4 раза: условие выполнено для значений i 0, 1, 2 и 3. Надо заменить границу с 4-ки на 3-ку

i = 0
while i < 3:
    print('Всё работает как надо')
показать ответ
Строка будет выводится до бесконечности, т.к. не увеличивается итератор и условие всегда выполнено, надо добавить строку i = i + 1

i = 0
while i < 3:
    i = i + 1
print('Всё работает как надо')
показать ответ
Забыт отступ перед строкой, с командой print, поэтому строка вне цикла и будет выведена единожды

i = 0
while i < 1:
    i = i + 1
    print('Всё работает как надо')
while i < 2:
    i = i + 1
    print('Всё работает как надо')
показать ответ
cтрока будет выведена дважды: 1 раз в первом цикле для i = 0, а затем ещё один раз во втором цикле, ведь мы не обнулили i и она будет равна 1 при заходе, нужно либо исправить границу во втором цикле на 3, либо добавить обнуление переменной-итератора i = 0
